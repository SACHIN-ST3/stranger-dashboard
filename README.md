# STRANGER — Local Dashboard

A cleaned and split version of the Claude Artifact you shared.

## Files
- `index.html` — structure
- `css/styles.css` — all styling
- `js/app.js` — all behavior

## Run it locally
Just open `index.html` in your browser.

## Notes
- Data is stored in `localStorage`
- Profile photos, cover photos, and posts are saved as data URLs
- The vault is a local demo and stores data in your browser only
- Requires internet for Google Fonts, Tabler Icons, and Chart.js CDN assets
